## To run locally

1. clone repo   
```
git clone https://github.com/bislara/project-enigma.git
```

2. cd into directory   
```
cd project-enigma
```

3. install dependencies using npm
```
npm i
```
3. install dependencies using npm 
```
npm start
```

Note:  Please use npm and "not" yarn. The deployment configuration is set using npm.
